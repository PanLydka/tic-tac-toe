## Live Version

Here: https://panlydka.github.io/tic-tac-toe-ES5/

## Installing

```
git clone https://github.com/PanLydka/tic-tac-toe-ES5
cd tic-tac-toe-ES5
npm install
gulp /or/ npm run gulp
```
